Welcome to ArdDroid.

Basically, with this project me and my associate are trying to do simple connections between Arduino and Android. With this, our objective is to create an Android App with full control over an Arduino Board.


In this version, we only got a basic app that is used to control LEDs present in Arduino Uno. - the app are not finished, so, there is a long way to go.
